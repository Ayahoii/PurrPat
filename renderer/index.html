🎭 Como criar expressão WebItemAction

Nenhuma expressão do tipo WebItemAction foi encontrada no modelo atual. Siga os passos abaixo para criar uma:

  1. Spawne um Web Item no VTube Studio.
  2. Clique na engrenagem do Web Item. Em Initial URL, coloque a URL do overlay configurada neste app: http://localhost:3000/patpat Clique em OK.
  3. Vá em Hotkeys Settings & Expressions.
  4. No final da lista de hotkeys, clique em +.
  5. Em Escolher Ação, selecione Trigger Web Item Action.
  6. Defina a expressão e escolha Spawn new.
  7. Renomeie a expressão para PatPat para facilitar identificação.

💡 Depois de criar, clique em Atualizar na lista de expressões para ela aparecer aqui.
